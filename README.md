# Python Advanced Gists

Продвинутые темы из Python.

## Метаклассы

Применение метаклассов на примере шаблона проектирования Singleton. 

Запуск:
```bash
python -m venv .venv

# Linux
source .venv/bin/activate
# Windows
.venv/Scripts/activate

pip install -r requirements.txt

python ./singleton.py
```
